# EasyPaste 安装说明

这是一个本地剪贴板历史工具。

## 使用前先知道

- 历史记录只保存在本机，不上传网络。
- 第一次使用时，macOS 可能会拦截打开，这是正常现象。
- 想用“回车后自动切回原应用并粘贴”，必须开启“辅助功能”权限。
- 如果不授权，也可以正常复制到剪贴板，只是不会自动帮你粘贴。

## 安装步骤

1. 解压收到的压缩包。
2. 把 `EasyPaste.app` 拖到“应用程序”目录，或者放在你习惯的位置。
3. 第一次打开时，如果系统提示无法验证开发者：
   - 右键 `EasyPaste.app`
   - 选择“打开”
   - 再点一次“打开”
4. 如果还是被拦：
   - 打开“系统设置 -> 隐私与安全性”
   - 在底部找到 EasyPaste
   - 点击“仍要打开”

## 开启自动回填

1. 打开 EasyPaste。
2. 在主界面或菜单栏里点击“打开设置”。
3. 进入“系统设置 -> 隐私与安全性 -> 辅助功能”。
4. 把 `EasyPaste` 勾选上。
5. 回到 EasyPaste，点击“刷新状态”。

## 基本使用

1. 用全局快捷键呼出 EasyPaste。
2. 输入关键词搜索历史内容。
3. 用上下方向键切换记录。
4. 按回车：
   - 已授权：会尝试切回原应用并粘贴
   - 未授权：只会复制到剪贴板

## 常见问题

### 1. 为什么按回车没自动粘贴？

通常是没有开启“辅助功能”权限。

### 2. 为什么更新新版本后又要重新授权？

因为当前版本不是正式签名分发版，macOS 有时会把新包当成新的应用。

### 3. 为什么系统会提示来源不明？

因为这是朋友间试用版，不是 Apple Developer Program 的正式签名和公证版本。
